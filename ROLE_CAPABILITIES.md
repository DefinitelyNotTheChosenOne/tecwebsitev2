# 🏛️ Role-Specific Functionality Manifest

## 👤 USER (STUDENT) FUNCTIONS
*The Student Experience focus on discovery and academic growth.*

*   **Marketplace Discovery**: Browse the "Academic Market" (dynamic subject list filtered by availability).
*   **Specialist Search**: View deep-dive public portfolios of Tutors.
*   **Help Wanted Postings**: Create public help requests to attract specialists.
*   **Mission Control Dashboard**: Track all active and pending tutoring requests.
*   **Negotiation Tunnel**: Real-time chat with Tutors to discuss pricing and schedule.
*   **Safety Reporting**: Select specific chat messages to flag and report to High Command.
*   **Live Class Terminal**: Access a time-locked, distraction-free environment for lessons.
*   **Mission Archive**: View history of all completed classes.

---

## 🛠️ SELLER (SPECIALIST) FUNCTIONS
*The Specialist Experience focuses on professional management and fulfillment.*

*   **Dossier Management**: Sculpt public bio, profile, and specialized skill tags.
*   **Credential Vault**: Securely upload resumes/transcripts for student verification.
*   **Mission Scanning**: Receive automated recommendations of Student requests matching your skills.
*   **Negotiation Interface**: Manage incoming student leads and finalize class terms.
*   **Schedule Deployment**: Set and manage the exact dates/times for tutoring sessions.
*   **Execution Terminal**: Real-time chat and interaction tools inside live class sessions.
*   **Earnings Tracking**: (Database-ready) Monitor completed sessions and performance stats.

---

## ⚡ ADMIN (HIGH COMMAND) FUNCTIONS
*The Admin Experience focuses on platform integrity and system governance.*

*   **Global Taxonomy Control**: Deploy and manage the master "System Subjects" list.
*   **Smart Filter Logic**: Automatically control which subjects appear on the marketplace based on seller density.
*   **Dispute Resolution Terminal**: Intervene in stuck or contested classes; release or refund escrow funds.
*   **Moderation Command**: Review "Flagged Content" reports submitted by students.
*   **User Registry**: Oversight of all accounts, Strike status, and credential verification.
*   **System Analytics**: Monitor transaction velocity, active user loads, and server health.
*   **Support Intercept**: Directly communicate with users in secure support tunnels.
